# E-Commerce Data Analytics and Preprocessing System

## Overview

This project was developed as part of a Data Analytics training program to demonstrate the complete workflow of data preprocessing, feature engineering, and exploratory data analysis on an E-Commerce dataset. The primary objective of the project is to transform raw transactional data into a clean, structured, and analysis-ready format while uncovering meaningful business insights.

The project showcases essential data analytics techniques, including data cleaning, missing value treatment, outlier detection, feature engineering, and statistical analysis using Python and industry-standard data analysis libraries.

## Features

* Data cleaning and preprocessing
* Missing value handling
* Duplicate record removal
* Outlier detection and treatment using the IQR method
* Feature engineering for enhanced analysis
* Statistical and descriptive analysis
* Customer purchasing behavior analysis
* Cleaned dataset generation for future analytics and machine learning tasks

## Technologies Used

* Python
* Pandas

* SciPy

* Jupyter Notebook

## Data Preprocessing Implemented

### Missing Value Handling

Missing values were identified and treated using appropriate statistical techniques such as median and mode imputation.

### Duplicate Removal

Duplicate records were detected and removed to maintain data quality and consistency.

### Outlier Treatment

Outliers were identified using the Interquartile Range (IQR) method and capped to minimize their impact on analysis.

## Feature Engineering

The following analytical features were created:

### Order Month

Extracts the month from the order date.

### Order Year

Extracts the year from the order date.

### Average Item Price

Calculates the average price per purchased item.

### Cart Value

Computes the total estimated cart value.

### Coupon Usage Indicator

Identifies whether a coupon was applied to an order.
